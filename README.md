# UltrAspire-Lumen-850-Duo-cap

This is a OpenSCAD file and associated .stl to 3D print a protective cap for the UltrAspire Lumen 850 Duo waist light.  This cap will click on to protect the on-off switch of the light from being accidentally turned on during transport or other times when not in use.

Feel free to use as necessary.

You can find OpenSCAD here: https://openscad.org
